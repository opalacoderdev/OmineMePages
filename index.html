Designed for Developer Autonomy

OmniMe brings the power of state-of-the-art coding assistants directly to your terminal and local IDE setup.

MemGPT Memory

Persistent SQLite-backed project memory so the agent never forgets your past choices, architecture decisions, or configuration settings.

Modular Skills System

Opt-in capabilities managed via a local skills.yaml. Spawns specialized, focused sub-agents to execute web searches, refactoring, or custom scripts.

Shadow Git VCS

Automatic checkpoints of your workspace to an isolated directory before execution. Safely rollback changes instantly at any time using /undo.

React & Vite GUI IDE

A beautiful web-based interface powered by pywebview. Built-in xterm.js terminal, Git source control sidebar, and global settings toggles.

Ollama Optimization

Engineered to extract maximum performance from local LLMs. Run open-weights models like Gemma 12B completely offline on your GPU.

Zero-Configuration Privacy

Complete control over your proprietary code. Keep your codebase on-premise and local without sending data to third-party endpoints.

Under the Hood Architecture

OmniMe coordinates memory, tools, and ephemeral agents to execute workflows cleanly.

Quick Installation & Startup

Deploy OmniMe to your system and configure your first autonomous project workspace.

Download the precompiled, standalone desktop application for your operating system:

Download for Windows (.exe) Download for Linux (AppImage)

Ensure Python 3.11+ is installed. Execute the command below to fetch the agent block configuration and start the main loop:

pip install omnime

Clone the repository and install requirements. Recommended for local adjustments or developer integrations:

git clone https://github.com/opalacoderdev/OmniMe.git
cd OmniMe
python -m venv .env
source .env/bin/activate
pip install -r requirements.txt

Launching the GUI

Once installed, start the React desktop application via Pywebview (with automatic fallback to a local server) by calling:

python main.py
